Usando a tag : A tag é a maneira mais simples de incluir um arquivo CSS. Essa é a forma preferida de usar o CSS como ele se destina a ser usado: um arquivo CSS é incluído por todas as páginas de seu site, e mudar uma linha nesse arquivo afeta a apresentação de todas as páginas do site. Usando a tag de estilo (style): Em vez de usar a tag de para apontar para uma folha de estilo separada contendo o arquivo CSS, podemos adicionar o CSS diretamente dentro de uma tag de estilo.
